🚀 App Store 출시 가이드

앱을 처음 출시하는 사람을 위한 단계별 체크리스트입니다.
위에서부터 순서대로 따라가세요.

Phase 1 출시 전 준비
  • Apple Developer Program 가입 (연 $99, 약 13만 원)
  • App Store Connect 접속 확인 (appstoreconnect.apple.com)
  • 번들 ID 결정 (com.yourname.appname 형식)
  • 앱 이름 결정 — App Store에서 검색될 이름 (30자 이내)
  • 부제목 결정 (30자 이내, 핵심 기능 한 줄)
  • 카테고리 선택 (주 카테고리 + 보조 카테고리)
  • 개인정보 처리방침 URL 준비 (필수. 간단한 페이지라도 필요)
💡 개인정보 처리방침은 GitHub Pages에 마크다운 한 페이지로 만들어도 충분합니다.
Phase 2 앱 빌드 & 업로드
  • Xcode에서 Signing & Capabilities 설정 (Team, Bundle ID 확인)
  • 앱 아이콘 준비 (1024×1024px, 투명 배경 불가)
  • Launch Screen 설정
  • 버전 번호 설정 (1.0.0 권장)
  • 빌드 번호 설정 (1부터 시작)
  • 시뮬레이터가 아닌 실제 기기에서 테스트
  • Product → Archive → Distribute App → App Store Connect 업로드
  • App Store Connect에서 빌드가 "처리 중" → "준비됨"으로 바뀌는지 확인 (보통 10~30분)
💡 Archive가 안 되면 대부분 Signing 문제입니다. Automatically manage signing이 켜져 있는지 확인하세요.
Phase 3 App Store 메타데이터
  • 스크린샷 준비 — 최소 6.7" (iPhone 15 Pro Max) 사이즈 필수
  • 스크린샷 3~5장 (첫 번째가 가장 중요 — 핵심 기능을 보여줄 것)
  • 앱 설명 작성 (첫 3줄이 접힌 상태에서 보이는 전부. 여기에 핵심 가치)
  • 키워드 입력 (100자, 쉼표 구분. 앱 이름에 포함된 단어는 중복이니 빼기)
  • 지원 URL 입력 (개인정보 처리방침과 같아도 됨)
  • 연령 등급 설정 (설문 응답)
  • 가격 설정 — 무료 or 유료
  • 출시 국가/지역 선택
💡 스크린샷은 실제 스크린샷보다 "기능 설명이 들어간 마케팅 이미지"가 다운로드를 더 많이 만듭니다. Figma나 Canva로 만들 수 있습니다.
Phase 4 심사 제출
  • 빌드 선택 (업로드한 빌드를 메타데이터에 연결)
  • 심사 메모 작성 — 로그인이 필요하면 테스트 계정 정보 제공
  • "심사를 위해 제출" 클릭
  • 상태가 "심사 대기 중"으로 변경되는지 확인
💡 첫 심사는 보통 24~48시간 걸립니다. 주말에 제출하면 더 걸릴 수 있습니다.
Phase 5 리젝 대응

리젝은 실패가 아닙니다. 첫 출시에서 한 번은 거의 무조건 됩니다. 당황하지 마세요.

  • 리젝 사유 읽기 — Resolution Center에서 구체적 이유 확인
  • 가이드라인 번호 확인 (예: 2.1, 4.3 등)
  • 해당 가이드라인을 Apple 문서에서 읽기
  • 수정 후 새 빌드 업로드
  • Resolution Center에서 수정 내용 설명 후 재제출

자주 걸리는 리젝 사유 TOP 5

  • Guideline 2.1 — 성능
    크래시, 버그, 빈 화면. 실기기에서 충분히 테스트하면 피할 수 있음
  • Guideline 4.3 — 스팸/복제
    다른 앱과 너무 비슷하거나, 기능이 너무 단순한 경우. "이 앱만의 가치"를 심사 메모에 설명
  • Guideline 5.1.1 — 개인정보
    개인정보 수집하면서 처리방침이 없거나 부족. Privacy Manifest 확인
  • Guideline 3.1.1 — 인앱결제
    외부 결제 유도 금지. 디지털 콘텐츠는 반드시 Apple IAP 사용
  • Guideline 2.3 — 메타데이터
    스크린샷이 실제 앱과 다르거나, 설명이 오해를 주는 경우
⚠️ 리젝 사유를 이해 못 하겠으면 위클리 핫시트에서 같이 봅시다. Resolution Center 스크린샷을 캡처해오세요.
Phase 6 출시 완료 후
  • 앱이 "판매 준비됨" 상태인지 확인
  • App Store에서 직접 검색해서 다운로드 테스트
  • 주변 3~5명에게 공유하고 피드백 수집
  • 크래시 리포트 확인 (Xcode Organizer 또는 App Store Connect → Analytics)
  • 첫 리뷰가 달리면 답글 달기
  • 축하합니다 🎉 — 주간 체크인에 출시 완료라고 적어주세요
